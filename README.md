Projeto Módulo 27 do Curso de Ciência de Dados EBAC

Objetivo: utilizar esses dados históricos dos pacientes e construir um bom modelo de regressão capaz de indicar se novos pacientes estão propensos a doenças cariovasculares ou não.

Chegamos a conclusão que doenças cardíacas dependem de muitos fatores que podem não estar na tabela (genética, estresse, sono).
